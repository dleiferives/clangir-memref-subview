# ClangIR: Memref Subviews 

While ClangIR has been integrated into the llvm/llvm-project repository. And all future development will take place there.

This project is the closest way that we so far have to getting C/C++ -> MLIR with memref subviews. This is not an official ClangIR implementation. I just need some memref subviews from C code. So I've split off the dead branch and will and will develop that behaviour here for the moment-- at least untill more upstreaming work is done.
